# sentiment-analysis-social-media-mentions
This repository contains the development of an automated system designed to analyze and classify sentiments from social media mentions to support data-driven business decisions.

## Project Overview
The primary goal of this project was to create a pipeline that identifies and categorizes public sentiment (Positive, Negative, Neutral) regarding specific brands or topics on social media. By transforming raw social media data into structured insights, the project helps organizations monitor their reputation and respond to customer trends.

## Key Features
* Data Engineering: Performed extensive cleaning and preprocessing on social media datasets to remove noise and special characters.
* Database Design: Developed a structured business database to store processed data and provide actionable insights for decision-making.
* Integration: Focused on ensuring data integrity through a well-defined schema designed for long-term scalability.
* Documentation: Includes a user manual and logic mapping for the sentiment classification workflow.

## Technologies Used
* Languages: Python (Pandas, Scikit-learn, NLTK).
* Database: SQL / Structured Business Database.
* Version Control: Managed development through modular Git branches

## Repository Structure
* /data: Contains processed and cleaned datasets used for sentiment classification.
* /docs: Final project documentation and user manuals.
* /src: Modular code for data processing and model logic.
